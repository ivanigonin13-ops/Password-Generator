## 🛠️ Как установить и запустить (Инструкция для всех ОС)

Перед началом убедитесь, что на вашем компьютере установлен **Python** (версии 3.8 или новее). Если его нет, скачайте его с [официального сайта python.org](https://python.org). При установке на Windows обязательно поставьте галочку **"Add Python to PATH"**.

### Шаг 1. Скачивание программы

Вы можете получить проект двумя способами:

* **Вариант А (Для разработчиков):** Склонируйте репозиторий через Git, выполнив команду в терминале:
  ```bash
  git clone https://github.com/ivanigonin13-ops/Password-Generator.git
  cd smart-downloads-organizer
  ```

* **Вариант Б (Для обычных пользователей):** Нажмите зеленую кнопку **Code** вверху этой страницы на GitHub, выберите **Download ZIP** и распакуйте скачанный архив в любую удобную папку (например, на Рабочий стол).

---

### Шаг 2. Установка и Запуск

Выберите инструкцию для вашей операционной системы:

#### 🪟 Инструкция для Windows
1. Откройте папку с распакованной программой.
2. Кликните по пустой строке с адресом папки вверху проводника, введите три буквы `cmd` и нажмите **Enter**. Откроется черное окно (Командная строка).
3. Установите необходимые компоненты, введя команду и нажав Enter:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите программу:
   ```bash
   python generator.py
   ```
5. Программа начнет работу! Чтобы выключить её, просто закройте черное окно или нажмите `Ctrl + C`.

#### 🍏 Инструкция для macOS
1. Откройте программу **Терминал** (через поиск Spotlight или в папке Программы -> Утилиты).
2. Наберите слово `cd ` (с пробелом на конце), перетащите папку с программой из Finder прямо в окно Терминала и нажмите **Enter**.
3. Установите необходимые компоненты:
   ```bash
   pip3 install -r requirements.txt
   ```
4. Запустите программу:
   ```bash
   python3 generator.py
   ```
5. Чтобы остановить программу, нажмите комбинацию клавиш `Control + C` в Терминале.

#### 🐧 Инструкция для Linux
1. Откройте терминал и перейдите в папку с проектом:
   ```bash
   cd /путь_к_папке/Password_gen
   ```
2. Установите зависимости:
   ```bash
   pip3 install -r requirements.txt
   ```
3. Запустите скрипт:
   ```bash
   python3 generator.py
   ```
4. Для остановки трекера нажмите `Ctrl + C`.
